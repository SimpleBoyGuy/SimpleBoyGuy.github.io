APDPO for FJSP

Reinforcement Learning for the Flexible Job Shop Scheduling Problem

This project studies how reinforcement learning can be used to solve the Flexible Job Shop Scheduling Problem (FJSP), with the goal of producing scheduling policies that are both high-quality and practical for complex manufacturing decision-making.

Project Overview

The Flexible Job Shop Scheduling Problem extends classical job shop scheduling by allowing each operation to be processed on multiple candidate machines. This flexibility makes the problem much more realistic for practical production systems, but also significantly increases the complexity of making good scheduling decisions.

In this project, I explore reinforcement learning as a way to learn scheduling policies directly from problem instances and system feedback, instead of relying only on handcrafted dispatching rules or expensive iterative optimization procedures.

Problem Description

In FJSP, each job consists of a sequence of operations with precedence constraints. For every operation, there may be several compatible machines, and the processing time can vary across machines. A solver therefore needs to determine not only which operation to schedule next, but also which machine should process it.

This creates a difficult combinatorial decision problem because sequencing and machine assignment are tightly coupled. The search space grows rapidly as the number of jobs, operations, and machine choices increases, making high-quality and efficient decision-making especially challenging.

Why Reinforcement Learning

Traditional approaches to FJSP often face a trade-off. Exact optimization methods can be accurate but computationally expensive, while heuristic and metaheuristic methods can be faster but may not always provide consistently strong solutions. Reinforcement learning offers a different perspective: learning a policy that can make sequential scheduling decisions adaptively from state information.

My goal is not only to obtain strong scheduling performance, but also to build RL-based solvers that are interpretable, efficient, and capable of supporting practical decision-making settings.

APDPO

My CDC 2025 work introduces Action-Priority Driven Policy Optimization (APDPO), a reinforcement learning method designed for FJSP. The method is motivated by three main goals: improving solution quality, increasing interpretability through a more intuitive policy representation, and reducing computational cost through direct policy updates.

  • Action-priority policy representation for more transparent scheduling decisions
  • Direct, gradient-free policy updates for improved computational efficiency
  • A reinforcement learning framework tailored to the structural characteristics of FJSP

Current Focus

The broader project aims to develop a reinforcement learning solver for scheduling problems that is not only academically strong, but also meaningful for real-world applications. I am especially interested in connecting rigorous algorithmic ideas with practical scheduling environments where decision quality, efficiency, and scalability all matter.
